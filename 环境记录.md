## 🖥️ 系统信息

- **设备**: Mac M2 (Apple Silicon)
- **系统**: macOS (Darwin 23.5.0)
- **内存**: 16 GB
- **Python**: 3.11.3
- **PyTorch**: 2.0.0
- **MPS**: ✅ 可用

## 🎯 项目目标

**核心目标**: 使用 IndexTTS2 为视频快速生成配音，替代1-2小时的人工录制

**使用场景**: 家政保洁服务 → 抖音短视频 → 获取客户

**工作流程**: 文案 → IndexTTS2配音（5-10分钟） → 剪映编辑 → 抖音发布

## 🚀 下一步行动

1. **安装依赖**
   ```bash
   pip install librosa soundfile
   pip install -r requirements.txt
   ```

2. **准备声音样本**
   - 录制5-10分钟清晰声音
   - 保存为WAV格式
   - 放到 `data/reference/` 目录

3. **获取模型**
   - 下载预训练模型，或
   - 使用声音样本进行微调

4. **测试配音生成**
   ```bash
   python scripts/video_dubbing/dubbing_for_video.py \
       --text "测试文案" \
       --reference_audio data/reference/my_voice.wav \
       --output outputs/audio/test.wav \
       --emotion happy \
       --checkpoint checkpoints/pretrained/model.pth
   ```
