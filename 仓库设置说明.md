## 📝 常用 Git 命令

### 日常使用
```bash
# 查看状态
git status

# 添加文件
git add <文件名>
git add .  # 添加所有更改

# 提交更改
git commit -m "提交说明"

# 查看提交历史
git log --oneline

# 查看差异
git diff
```

```bash

# 拉取远程更新
git pull origin main

# 查看远程信息
git remote -v
```
